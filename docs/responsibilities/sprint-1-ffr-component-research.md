## Overview

Research all available FFR components and modules. Determine which can be downloaded and simulated locally, which have open-source or printable alternatives, and which can be manufactured at MSU via CAVS. The goal is a categorized list of what physical and virtual assets the team can access for development and testing.

## Relevant Information

- FFR separable interfaces: CrossLink Cleat, iSSI, FuseBlox — dimensions and specs available
- FFR robotic arm: 7-DoF, Motiv Space Systems — sim package availability TBD
- CAVS manufacturing: CNC (6/7-axis), metal AM (Ti-6Al-4V, AlSi10Mg), waterjet, electronics
- Open-source robotics frameworks: ROS2, MoveIt2, Gazebo
- Motiv Guest Roboticist program provides sim tools — not yet requested

## Deliverables

- [ ] Catalog all known FFR components and modules from available documentation
- [ ] Categorize each component: downloadable or simulatable, open-source printable, manufacturable at CAVS
- [ ] Identify any components requiring vendor NDA or license
- [ ] Produce a tiered component availability list as reference for concept development
